Catch Up!

Github


Catch-Up is a quote management system that I developed with Mason Lee. We made Catch-Up as we wanted a better way to manage funny things our friends said that we wanted to remember. There are several layers of software in the project including a Discord bot, an API, a database, and a web app.


It was made with Django, MySQL, as well as Discord.py. Additionally it has a single sign-on authentication system to allow users to login with their Discord accounts.
